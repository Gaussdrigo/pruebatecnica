Prueba Técnica CLT
Backend

Spring Boot 3.5

JDK 17

Base de Datos: PostgreSQL 16 (SQL)

Tener creada una base de datos local con el nombre: clientdb

Frontend

Node 20.0.0

Next.js 15.2

Compilación y ejecución
1️⃣ Compilar Backend
mvn clean install
2️⃣ Levantar Backend
mvn spring-boot:run
3️⃣ Levantar Frontend
npm run dev
